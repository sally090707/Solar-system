# 太阳系天体系统

这是一个使用Three.js构建的3D太阳系模拟项目。太阳位于中心，行星围绕太阳旋转。

## 功能

- 太阳：黄色球体
- 行星：水星、金星、地球、火星、木星、土星、天王星、海王星，每个有不同的颜色、大小和轨道速度
- 动画：行星持续围绕太阳旋转

## 如何运行

1. 打开 `index.html` 文件在浏览器中。
2. 享受3D太阳系动画。

## 技术栈

- Three.js (通过CDN加载)
- HTML5 Canvas
- JavaScript

## 文件结构

- `index.html`: 主HTML文件
- `script.js`: Three.js代码
- `.github/copilot-instructions.md`: Copilot自定义指令
